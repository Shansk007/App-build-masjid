# 🕌 Masjid Prayer App (Kivy + Android APK)

A beautiful mobile application to manage **5-time prayer schedules for multiple Masjids**, built using **Python (Kivy/KivyMD)** and packaged into an **Android APK using Buildozer**.

---

## 🚀 Features

- 📍 Add multiple Masjids
- 🕒 Store 5-time prayer timings
- 📋 View Masjid details
- 💾 Data stored locally (JSON)
- 📱 Android APK support
- 🎨 Clean UI using KivyMD

---

## 🛠️ Tech Stack

- Python 3.10
- Kivy / KivyMD
- Buildozer
- Python-for-Android
- Android SDK / NDK

---
